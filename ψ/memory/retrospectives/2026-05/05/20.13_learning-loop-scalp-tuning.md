## Lessons Learned

1. **Stacked filters compound exponentially**: 6 filters each letting ~60% through = 0.6^6 = 4.7% pass rate. Always calculate the composite pass rate when designing multi-stage pipelines.
2. **Professional scalping consensus on XAUUSD**: M5 is the sweet spot (not M1), NY session is essential, ADX threshold for M1 should be 10 not 20, and direction threshold should be 0.15 not 0.30.
3. **Bridge API discovery**: Always run `dir(conn.root)` on rpyc connections before calling methods. The exposed_ prefix convention and init+login requirement are not always documented.
