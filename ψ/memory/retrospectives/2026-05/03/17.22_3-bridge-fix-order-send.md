## Lessons Learned

1. **Wait for background processes before modifying their output** — When gmag11 runs in background, you must `wait` for its PID before touching its files. Polling for a file is insufficient because the process continues creating files after the file appears.
2. **RPyC netref dicts support only bracket access, not `.get()` or `dict()` construction** — This is the third time we hit this. It's now documented in our learnings.
3. **MT5 `order_send()` requires the request dict constructed inside Wine Python** — Passing a dict from macOS via RPyC doesn't work because MT5 expects a local Python dict, not a netref proxy. Use explicit function arguments instead.
4. **XAUUSDm filling_mode = 3 (supports both FOK and IOC)** — Symbol info confirms this. Use type_filling=0 (FOK) for Exness.
5. **"Market closed" (retcode 10018) is the correct response on Sunday** — Not an error, just the market status.
