## Lessons Learned

1. **Check the simplest thing first** — When VPS showed wrong config, the .env file itself was the root cause. Docker-compose env propagation is straightforward; always verify the source file before debugging the pipeline
2. **LEARNING_MODE = vibe trading is a baseline shift** — This is the first time we're running with learning mode active. All future trading results need to be compared against this baseline. Before vibe trading: WR ~38-39%, PnL positive due to R:R. After learning loop kicks in: need to track if WR improves
3. **Cloud LLM models outperform local for analysis** — glm-5.1:cloud gives deeper, more actionable analysis than qwen3:8b. The 8/10 vs 7/10 score isn't just about number — the suggestions (Monte Carlo, trailing stop, blended approach) are more sophisticated
