## Lessons Learned

**Lesson**: XAU/USD requires wider stops than forex. ATR multiplier of 2.5 consistently outperforms 1.5 for gold because gold's daily range (1500-3000 pips) means tight stops get clipped. This aligns perfectly with the user's best practice recommendation of ATR 1.5-3.0x for SL.

**Lesson**: Config-code disconnect is the #1 architectural debt. Three YAML files with good values exist but no code reads them. This is the single highest-impact fix because it enables all subsequent parameter tuning without code changes.

**Lesson**: No single indicator combination achieves >50% WR on gold with the current approach. The missing ingredients are: (1) macro correlation (DXY/US10Y), (2) news filtering, (3) RSI 80/20 for overextension detection, and (4) EMA200 hard direction filter. These are XAU/USD-specific and forex systems don't need them.
